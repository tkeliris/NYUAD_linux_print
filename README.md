# NYUAD_linux_print
New York University (Saadiyat island campus) pharos printer support for Linux

##README
Printing on NYU Abu Dhabi printers from Ubuntu (Deb) machines.

##Install
Straightforward, just ```source NYUAD_UNX_PRINT.sh```
